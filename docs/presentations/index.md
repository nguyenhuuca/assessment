# Presentations

A curated set of **stakeholder-ready presentations** for the Funny Movies
platform. Each deck distills a product or engineering document (PRD, Spec, ADR,
or Plan) into a self-contained, full-screen slide experience — designed to walk
business and technical stakeholders through *what* we are building, *why*, and
*how*.

!!! tip "How to view"
    Open a deck from the **Presentations** menu (left). Each one opens as an
    interactive tab-bar deck — click the tabs or use <kbd>←</kbd> / <kbd>→</kbd>
    to move between sections, and the 🔎 toggles to expand details.

## Why these exist

- **One source of truth** — every deck is generated directly from its underlying
  document, so the slides never drift from the spec.
- **Audience-friendly** — dense docs become a guided narrative: problem → goals →
  scope → requirements → risks.
- **No setup** — each deck is a single HTML file; it opens in any browser, no
  build step or tooling required.
